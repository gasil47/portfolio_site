Kinetic LED Control

Unity-based multi-device control system for kinetic LED displays

Overview

This Unity-powered system enables synchronized control of kinetic LED display movements and visuals using a multi-device architecture. One instance can run on a tablet, allowing users to interact with visual controls, while another Unity instance communicates directly with the kinetic LED display hardware.

Key Features

  • Real-time device-to-device data sync
  • Interactive tablet UI for controlling visuals
  • Custom Unity TCP/UDP communication layer
  • Supports various kinetic LED hardware protocols
  • Responsive and intuitive user feedback loop
