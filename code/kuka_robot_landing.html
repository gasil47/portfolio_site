KUKA Robot Sync Tool

Real-time communication between Unity and KUKA KR210 R2700

Overview

This Unity-based application establishes a real-time link between a procedural digital twin and a physical KUKA KR210 R2700 robot using the C3 Bridge protocol. It supports bi-directional streaming of joint and end-effector data, allowing live synchronization and control of robotic movements through Unityâ€™s interface.

Key Features

  • Live joint control and visualization from Unity
  • Real-time feedback from robot to Unity
  • Support for both E6AXIS and E6POS structures
  • Built-in TCP/IP communication with C3 Bridge protocol
  • Custom slider-based and IK-based controls
  • Visual Gizmo overlays for TCP pose
