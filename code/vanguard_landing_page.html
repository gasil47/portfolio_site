Vanguard

3rd person shooter game with military combat scenarios built in Unity

Overview

Vanguard is a third-person shooter game developed in Unity, offering a tactical experience featuring military enemies, diverse weapon systems, and immersive environments. The game includes sniper mechanics, gun recoil, and a fully implemented health and ammo system. It provides an engaging combat experience with cinematic cutscenes and mission-based progression.

Key Features

  • Third-person shooting mechanics with aim and cover
  • Multiple weapons: rifles, snipers, pistols
  • Ammo pickup and management system
  • Health bar and damage feedback
  • Military-style enemies with AI behavior
  • Cutscenes and mission sequences
