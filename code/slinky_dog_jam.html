Slinky Dog Jam

A fun arcade-style game with a stretchy dog and tricky levels

Overview

Slinky Dog Jam is hyper-casual game where players solve tangled slinky dogs. I focused on slinky stretching mechanics, obstacle logic, and level progression.

Key Features

  • Spline-based stretchy movement
  • Slinky obstacles
  • Unique visuals
  • Progressive level unlocking

Tools & Technologies

  • Unity (C#)
  • Spline Physics
  • Level Manager System
