Slinky Jam

Bounce and glide your way through puzzling challenges

Overview

Slinky Jam is a colorful physics-based mobile game where players control a slinky to jump, stretch, and compress through platforms and obstacles. I contributed to the slinky motion algorithm, environment hazards, and level progression.

Key Features

  • Realistic spring-like movement
  • Creative level design
  • Colorful visual style
  • Responsive touch-based input

Tools & Technologies
