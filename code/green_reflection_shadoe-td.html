Real-Time Reflection & Shadow

Interactive setup using TouchDesigner, Kinect, and NVIDIA Broadcast

Overview

This project showcases a real-time interactive reflection and shadow system built with TouchDesigner. Using a Kinect sensor and NVIDIA Broadcast for background segmentation, users experience dynamic lighting and visuals that react to body movement and space.

Demo Video

Key Features

  • Real-time reflections based on user body and motion
  • Dynamic soft shadows cast from user silhouette
  • Background removal with NVIDIA Broadcast
  • Skeleton tracking via Kinect sensor
  • Optimized for live installations

Tools & Technologies

  • TouchDesigner
  • Kinect Azure Sensor
  • NVIDIA Broadcast SDK
  • Python Scripting
