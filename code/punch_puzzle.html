Punch Puzzle

Pattern-based punching game featuring logic and fun interactions

Overview

Punch Puzzle is a unique mobile puzzle game where players must figure out patterns and punch mechanisms to solve challenging levels. I worked on the core logic systems for puzzle generation and win-state detection, levels, and handled transitions and animation polish.

Key Features

  • Grid-based puzzle solving
  • Simple controls with increasing challenge
  • Pattern recognition mechanics
  • Minimalist animations

Tools & Technologies
