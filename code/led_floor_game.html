LED Floor Tile Game

Interactive game using a physical LED tile grid controlled via Unity

Overview

This application enables gameplay through a real-world LED tile floor where players step on tiles to solve a color puzzle. I built this using Unity with real-time feedback from tile inputs. It supports both demo mode and interactive live play through touch or physical input.

Key Features

  • Real-time color-based puzzle interaction
  • Supports foot-based inputs on tile grid
  • Dynamic difficulty & safe-zone logic
  • Animated feedback using tile lights

Tools & Technologies

  • Unity (C#)
  • Art-net Protocol
  • Custom input simulation & communication layer
  • Real-time tile state visualizer
