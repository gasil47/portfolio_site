Traffic Escape

A logic-based traffic redirection puzzle game

Overview

Traffic Escape challenges players to redirect multiple vehicles to safe exits without crashes. I worked on the path logic, vehicle AI, and step-by-step hint system for each level.

Key Features

  • Grid-based directional traffic system
  • Incremental level difficulty
  • Hint and rewind tools for players
  • Minimalist visual design
