Trifold Controller

Interactive animation + content controller for trifold LED displays

Overview

The Trifold Controller is a desktop app designed to control large kinetic LED trifold displays. It can trigger animation states, control Resolume via OSC, and even respond to RF remote inputs through a serial COM port. Built for real-time stage performances, events, and installations.

Key Features

  • Trigger multiple trifold display animations
  • Send OSC messages to Resolume to control visual content
  • Read COM port input for remote RF control
  • Real-time responsive feedback and sync
  • User-friendly UI for quick scene selection

Demo Video

Tools & Technologies

  • Unity (C#)
  • OSC Protocol (Open Sound Control)
  • Serial Communication for COM Port
  • Custom animation system
