Taxi Please

A casual ride-hailing simulation with quirky characters and vibrant cityscapes

Overview

In Taxi Please, players navigate city streets to pick up and drop off passengers while dealing with traffic and time constraints. I implemented the character spawning, cab controls, and interaction system between rider and driver.

Key Features

  • One-tap control and responsive UI
  • Humorous passenger animations
  • City navigation and parking mechanics
  • Level progression system

Tools & Technologies

  • Unity (C#)
  • NavMesh Pathfinding
  • Mobile Input Controller
