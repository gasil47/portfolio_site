Photon TPS Multiplayer

Third-person multiplayer shooter built with Photon PUN

Overview

This is a third-person multiplayer shooting game currently in development, powered by Photon PUN. Players can create or join custom rooms and engage in online gunfights with synced health, ammo, and animations across clients. Built with cross-platform support, it runs smoothly on mobile and desktop.

Current Features

  • Create and join rooms via room ID
  • Room list UI with live join buttons
  • Multiplayer synced shooting and health system
  • Death animations synced across players
  • Ammo management and reloading logic
  • Mobile-friendly joystick and UI input

Screenshots

Tech Stack

  • Unity (C#)
  • Photon PUN 2
  • Mobile joystick input
  • SyncVar + RPC methods for state sync
  • Modular room system

Status & Roadmap
