Interactive Floor Mapping

Real-time touch-sensitive visual feedback using TouchDesigner

Overview

This TouchDesigner-based application transforms a floor projection area into a dynamic interactive experience. Using a camera feed and spatial mapping, users can interact with zones on the floor that change visuals based on motion, gestures, and timing logic. Designed for exhibitions, games, or interactive installations.

Key Features

  • Safe zone calibration and real-time feedback
  • Motion-triggered animations and color changes
