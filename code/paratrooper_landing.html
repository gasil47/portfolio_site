Paratrooper

Classic 2D shooter reimagined — inspired by the 1982 original

Overview

Paratrooper is a retro-style 2D shooter inspired by the original 1982 game. You must defend your base from waves of descending paratroopers and helicopters using a stationary gun turret. It’s a fast-paced arcade challenge with updated visuals and smooth gameplay tuned for modern devices.

Key Features

  • Classic arcade-inspired mechanics
  • Pixel-perfect enemy sprites
  • Endless wave survival mode
  • Score tracking with leaderboard
  • Mobile and desktop support
