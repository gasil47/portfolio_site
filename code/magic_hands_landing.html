MagicHands

Third-person magic shooter game made for Crazy Games

Overview

MagicHands is a third-person shooter where instead of traditional weapons, players wield elemental magic through their hands. Defeat waves of unique enemies using projectiles and abilities while exploring stylized arenas. The game is optimized for web and designed for engaging, fast-paced fun.

Key Features

  • Elemental magic projectiles: fire, ice, electric, and more
  • Multiple enemy types: melee, ranged, magic-wielding
  • Intuitive mobile-style twin stick controls
  • Stylized 3D visuals and VFX for spell feedback

Tools & Technologies

  • Unity (URP)
  • Shader Graph VFX
  • WebGL Deployment
  • Mobile-style twin-stick input system
