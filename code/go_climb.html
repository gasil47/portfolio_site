Go Climb

Vertical runner challenge with physics-based climbing mechanics

Overview

Go Climb is a mobile climbing game where players ascend walls, cliffs, and buildings using physics-driven mechanics. I worked on rope simulation, grip zone logic, and balancing difficulty for engaging vertical runs.

Key Features

  • Physics-based climbing and swing system
  • Level progression with height-based goals
  • Interactive grip points and boosters
  • Dynamic camera tracking
