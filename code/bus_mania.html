Bus Mania

A traffic control and passenger pickup game for Android

Overview

In Bus Mania, players manage a fleet of buses in a busy city and ensure they pick up passengers on time without collisions. I implemented the bus AI logic, route optimization, and UI for trip management.

Key Features

  • City-wide routing and scheduling system
  • Color-coded bus line identification
  • Interactive UI with dynamic updates
  • Multiple map levels and unlockables
