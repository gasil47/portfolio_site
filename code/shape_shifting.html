Shape Shifting

Dynamic transformation runner game for mobile platforms

Overview

Shape Shifting is a mobile endless runner where players must adapt to obstacles by switching shapes in real time. I handled shape-change transitions, obstacle logic, and performance tuning for smooth gameplay.

Key Features

  • Real-time shape transformation
  • Obstacle-based quick decision gameplay
  • Custom animation and visual feedback
  • Responsive mobile controls
