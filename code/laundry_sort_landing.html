Laundry Sort

Fun and satisfying laundry sorting puzzle game

Overview

Laundry Sort is a relaxing and addictive puzzle game where players must sort colored clothes into matching baskets. With intuitive drag-and-drop controls, vibrant visuals, and increasingly challenging levels, it's a perfect casual mobile experience.

Key Features

  • Simple drag-and-drop gameplay
  • Hundreds of sorting levels
  • Colorful, clean graphics
  • Relaxing background music and sound effects
  • Rewarding animations and game feedback

Tools & Technologies

  • Unity (C#)
  • Custom 2D animations
  • Mobile touch input system
  • Google Play integration
