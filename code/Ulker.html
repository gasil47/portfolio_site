Ülker Ramadan Dessert Creator

Unity app powered by Stable Diffusion for generating custom desserts

Overview

This Unity-based app was built for Ülker’s Ramadan campaign. It allows users to create dessert combinations and uses locally running Stable Diffusion to generate realistic outputs based on selected ingredients. Designed for kids and families during Iftar activations.

Key Features

  • AI image generation via Stable Diffusion
  • Interactive dessert builder
  • Arabic & English support
  • Print/share generated images via QR
  • Mobile-friendly UI with feedback system
